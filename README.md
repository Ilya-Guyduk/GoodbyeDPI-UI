
# GoodbyeDPI GUI

Современная графическая обёртка для [GoodbyeDPI](https://github.com/ValdikSS/GoodbyeDPI) — утилиты для обхода Deep Packet Inspection (DPI) и разблокировки сайтов.

![GoodbyeDPI GUI](screenshot.png)

## 📋 Особенности

- 🎨 **Современный интерфейс** в тёмных тонах с интуитивной навигацией
- 📦 **Автоматическая загрузка** GoodbyeDPI и WinDivert.dll с GitHub
- ⚙️ **Все параметры GoodbyeDPI** доступны через графический интерфейс
- 🎯 **Быстрые пресеты** для разных сценариев использования
- 📊 **Реальный вывод лога** работы GoodbyeDPI
- 💾 **Сохранение настроек** между запусками
- 🔒 **Требует права администратора** (запрашивает автоматически)

## 🚀 Быстрый старт

### Windows

1. Скачайте последнюю версию `GoodbyeDPI-GUI.exe` из [релизов](../../releases)
2. Запустите от имени администратора (запросит автоматически)
3. При первом запуске программа предложит скачать GoodbyeDPI
4. Выберите пресет или настройте параметры вручную
5. Нажмите "▶ Запустить"

### Запуск из исходного кода

```bash
# Клонируйте репозиторий
git clone https://github.com/yourusername/GoodbyeDPI-GUI.git
cd GoodbyeDPI-GUI

# Установите зависимости (только Python 3.8+)
# Дополнительные библиотеки не требуются — только встроенный tkinter

# Запустите
python goodbyedpi_gui.py
```

Пресеты

· 9 + DNS: С автоматическим перенаправлением DNS

❓ Часто задаваемые вопросы

Программа не запускается / требует права администратора

GoodbyeDPI работает на уровне сетевых драйверов и обязательно требует права администратора. При запуске программа автоматически запросит повышение прав.

Не работает обход блокировок

1. Убедитесь, что GoodbyeDPI и WinDivert.dll находятся в одной папке
2. Попробуйте разные пресеты (рекомендуется начать с 9 — рекомендуемый)
3. Проверьте лог вывода на наличие ошибок
4. Временно отключите антивирус (может блокировать WinDivert)

Антивирус ругается на WinDivert.dll

WinDivert — легитимный драйвер для перехвата сетевых пакетов. Добавьте папку с программой в исключения антивируса.

Как обновить GoodbyeDPI?

Нажмите кнопку ↓ рядом с путём к GoodbyeDPI.exe — программа автоматически скачает последнюю версию с GitHub.

Где хранятся настройки?

В файле goodbyedpi_gui_config.json в папке с программой. Можно редактировать вручную или через интерфейс.
